# Adding Tables

To add a new **Table** to your document, start by typing a forward slash ‘/’ to trigger a slash command, and then select the Table option from the dropdown menu.

![null](/api/images/screenshot-2024-02-01-140458-AwNz.png)After selecting the Table option from the editor dropdown, a blank table will appear in the editor.
