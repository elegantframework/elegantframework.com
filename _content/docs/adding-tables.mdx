## Adding Tables

To add a new **Table** to your document, start by typing a forward slash ‘/’ to trigger a slash command, and then select the Table option from the dropdown menu.

![null](/images/screenshot-2024-02-01-140458-AwNz.png)

After selecting the Table option from the editor dropdown, a blank table will appear in the editor.

![null](/images/screenshot-2024-02-01-140751-Q4ND.png)

Adding and Deleting Columns

To add or delete a column, click the top of the last column of the table to activate the **Columns** dropdown menu.

![null](/images/screenshot-2024-02-01-141126-EwNz.png)

## Adding and Deleting Rows

To add or delete a row from a table, click the left side of the last row of the table to activate the **Rows** dropdown menu.

![null](/images/screenshot-2024-02-01-141045-A5Nj.png)
