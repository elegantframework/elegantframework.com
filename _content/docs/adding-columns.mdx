Break up your content into left and right sections using Columns. This guide will show you how to add columns to your content using the text editor.

# Adding Columns

To add a new set of **Columns** to your document, start by typing a forward slash ‘/’ to trigger a slash command, and then select the Columns option from the dropdown menu.
