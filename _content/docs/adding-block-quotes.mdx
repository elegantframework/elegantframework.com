## Adding Block Quotes

To add a new set of **Block Quotes** to your document, start by typing a forward slash ‘/’ to trigger a slash command, and then select the Block Quotes option from the dropdown menu.

![null](/images/screenshot-2024-02-01-091920-MxNT.png)Type your quote into the editor, and apply any styling to the text that you would like.
