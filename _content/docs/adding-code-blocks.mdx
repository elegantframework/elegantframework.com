Add inline code blocks to your pages to set off your favorite example code or documentation. This guide will show you how to add Code Blocks to your content using the text editor.

 ## Adding a Code Block

To add a new **Code Block** to your document, start by typing a forward slash ‘/’ to trigger a slash command, and then select the Code Block option from the dropdown menu.

![null](/images/screenshot-2024-02-01-170340-Y4MT.png)After selecting the Code Block option from the dropdown, an empty code block will appear in the editor that you can fill with almost any type of code.
